analytics Document

    1. To Enable Analytics, add a key/value pair to your CampaignKit.plist file:

    “PKAnalyticsURL” = “https://analytics.radiusnetworks.com/sdk/v1/events”

    2. Optionally call [CKManager setPartnerIdentifier] when you create the CampaignKit manager:

    self.ckManager = [CKManager managerWithDelegate:self];
[self.ckManager setPartnerIdentifier:@"<id that represents a user here>"];
[self.ckManager start];

    2. Add calls to CKManager as needed to register events when the user takes action

    // record that this campaign has been viewed
[ckManager recordAnalytics:CKAnalyticsViewed forCampaign:campaign];
...
// record that this campaign has been fulfilled
[ckManager recordAnalytics:CKAnalyticsFulfilled forCampaign:campaign];
